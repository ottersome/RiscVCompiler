THIS IS THE ORIGINAL TABLES GIVE TO US BY TA
None of the methods are in the global scope method. This means that methods cannot call each other because the names are in other scopes in the same depth. 
For example: If we are using the proc tables and proc1 wants to call func1 then it cant. This is because 
    it is not in the global scope symbol table.
